# ふるさと納税 制度情報更新コマンド

docs/UPDATE_GUIDE.md のランブックD（告示改正の反映）を実行します。

## 実行内容

### ステップ1: 告示・法令改正を調査

WebSearchで以下を検索し、`docs/01_seido/rekishi.md` の年表に記録されていない新しい告示・法令改正がないか確認してください：
- 「総務省 ふるさと納税 告示 改正 最新」
- 「地方税法 ふるさと納税 改正 最新」
- 「e-Gov 平成31年総務省告示第179号 最新」

### ステップ2: 変更があれば反映

変更がある場合：
1. `docs/01_seido/kokuji.md` の該当基準（5割/3割/地場産品）を更新。改正日・改正内容を明記。
2. `docs/01_seido/rekishi.md` の年表に改正を1行追加。
3. ランブックCに従ってニュースメモ（archive）も作成。

### ステップ3: コミット

```
update: YYYY年◯月 告示改正（◯◯）を反映
```

変更がなかった場合は「告示改正なし。最終確認日: YYYY-MM-DD」と報告してください。
