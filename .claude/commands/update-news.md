# ふるさと納税 ニュース追跡コマンド

docs/UPDATE_GUIDE.md のランブックC（ニュース・制度改定の追跡）を実行します。

## 実行内容

### ステップ1: 直近のニュース・制度改定を調査

WebSearchで以下を検索し、`docs/02_news/timeline.md` に記録されていない新しいニュースを探してください：
- 「ふるさと納税 最新ニュース 制度改定」
- 「総務省 ふるさと納税 告示 通知 最新」
- 「ふるさと納税 楽天 訴訟 最新」
- 「ふるさと納税 2026」

### ステップ2: timeline.md を確認して未記録を特定

`docs/02_news/timeline.md` を読み込み、すでに記録済みの内容と照合してください。未記録のニュースのみを対象にします。

### ステップ3: 新ニュースをarchiveに記録

未記録のニュースごとに `docs/02_news/archive/YYYY-MM-DD_見出し.md` を新規作成：

```markdown
# YYYY-MM-DD ◯◯（見出し）

- 出典: （媒体名・URL・取得日）
- 種別: 告示 / 報道 / 統計発表 / 訴訟 / その他

## 要点（3行）
## 背景
## 解釈・影響
## 要確認（TODO）※あれば
## 関連リンク
```

### ステップ4: timeline.mdに追記

`docs/02_news/timeline.md` の先頭（最新順）に新しい行を追加：
```
| 日付 | 種別 | サマリ | archive リンク | 出典 |
```

制度本体に関わる改定なら `docs/01_seido/` の該当ファイルにも反映してください。

### ステップ5: コミット

```
news: ◯◯（見出し）を追記
```

新しいニュースがなかった場合は「新規ニュースなし。次回確認推奨: 1週間後」と報告してください。
