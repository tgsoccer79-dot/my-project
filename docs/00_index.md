## 学習ロードマップ（博士への道）

1. **基礎** — 制度の仕組み・歴史・税控除の計算（① 制度の体系知識）
2. **法令・基準** — 告示の文言を原文で読み、解釈できるようにする（①kokuji）
3. **時事** — 改定・報道を時系列で追い、背景まで説明できる（② ニュース）
4. **市場感覚** — ポータル・自治体・返礼品の売れ筋と寄附額の相場観（③ 市場）⭐️
5. **行財政** — 経費構造・収支・受益と負担のバランスを語れる（④ 経費・収支）
