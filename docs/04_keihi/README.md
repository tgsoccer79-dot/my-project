# ④ 自治体の経費・収支

どの自治体が・どんな経費で・どれだけ消耗しているかを整理する柱。

## このフォルダの構成

- [keihi_data.md](keihi_data.md) — 自治体別 経費・収支データ

## 経費の構成要素（告示の経費5割ルールと連動）

ふるさと納税の募集経費には主に以下が含まれる（詳細は [01_seido/kokuji.md](../01_seido/kokuji.md)）:

- 返礼品の調達費
- 返礼品の送料
- 広告・宣伝費
- ポータルサイト利用料（手数料）
- 決済手数料
- 事務費（受領証明書発行・問い合わせ対応 等）

## データの出典

- 総務省「ふるさと納税に関する現況調査結果」（経費率・経費内訳を公表）
- 各自治体の決算資料・公開情報
