# 宮崎県 都城市

> 最終更新: 2026-06-02

## 基本情報

- 都道府県 / 自治体名: 宮崎県 都城市（みやこのじょうし）
- 主産業: 畜産（畜産産出額 日本一クラス）・焼酎

## 戦略・特徴メモ

- 畜産産出額日本一を背景に「肉のまち」として圧倒的なブランド力。
- 肉・焼酎を軸に多数のポータルへ幅広く出品。切り落とし・大容量で件数を稼ぐ。
