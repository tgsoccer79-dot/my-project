## ポイント付与禁止（2025年10月1日施行）

### 概要

| 項目 | 内容 |
|------|------|
| 告示日 | 2024年6月28日 |
| 施行日 | **2025年10月1日** |
| 法的根拠 | 地方税法に基づくふるさと納税指定基準の改正告示 |
| 禁止の構造 | 「ポイントを付与するポータルを通じた寄附受付」を禁じる（自治体側の指定要件として課す） |

### 禁止対象 / 引き続きOKなもの

- **禁止**: 楽天ポイント・PayPayポイント等、ふるさと納税に伴う独自ポイント付与
- **OK**: クレジットカードの通常ポイント・決済ポイント（ふるさと納税以外でも付与される一般的なもの）

### 禁止の背景

- ポイント付与競争が激化し、自治体の費用負担が増大（手数料総額1,379億円・寄附総額の11.5%に相当）
- 「ポイント目的の寄附」が本来趣旨から乖離している、という総務省の判断

### 影響

- 2025年9月の駆け込み需要後、10月以降は市場の冷え込みが顕在化
- 自治体DX推進協議会調査：**91.1%の自治体が反動減を認識**、うち73.4%が「非常に大きな反動減」と回答
- RIETI分析：ポイント目当て利用者層の離脱が主因

出典:
- 総務省告示関連資料: https://www.soumu.go.jp/menu_news/s-news/01zeimu04_02000126.html
- 自治体DX推進協議会: https://prtimes.jp/main/html/rd/p/000000267.000132312.html
