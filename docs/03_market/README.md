# ③ 市場・返礼品データ ⭐️最優先

「どのポータルで・どの自治体の・どの返礼品が・どれだけ売れているか」「どの自治体がどれだけ寄附を集めているか」を整理する柱。

## このフォルダの構成

- [portals.md](portals.md) — ポータルサイト比較（楽天/さとふる/ふるなび/ふるさとチョイス 等）
- [ranking_kifu.md](ranking_kifu.md) — 自治体別 寄附額ランキング
- [ranking_henreihin.md](ranking_henreihin.md) — 人気返礼品ランキング（ポータル別・カテゴリ別）
- [jichitai_profiles/](jichitai_profiles/) — 注目自治体の個別プロファイル

## データ収集の方針

| 知りたいこと | 主な情報源 |
|--------------|-----------|
| 自治体別の寄附額・件数 | 総務省「ふるさと納税に関する現況調査」、各自治体の決算・発表 |
| 売れ筋返礼品 | 楽天ふるさと納税ランキング、さとふる・ふるなび・ふるさとチョイスの人気ランキング |
| ポータルのシェア・特徴 | 各社IR・プレスリリース、報道 |
| カテゴリ動向（肉/米/海産物/家電等） | ポータルのカテゴリ別ランキング、研究機関レポート |

## 収集時の注意

- **時点を必ず残す**: ランキングは時期で変動。`（YYYY-MM-DD取得／対象期間）` を明記。
- **ポータル横断で見る**: 同じ自治体でもポータルにより売れ筋が違う。比較軸を揃える。
- 構造化したい数値は `data/` にCSVでも持つ（Markdownは要約・解説、CSVは生データ）。
