## 指定制度の全体像（根拠法令の構造）

ふるさと納税の対象となる地方団体は、総務大臣が指定する（平成31年度税制改正・令和元年6月1日施行）。

- **地方税法 第37条の2第2項**（道府県民税側）／**第314条の7第2項**（市町村民税側）が、総務大臣の指定基準を3号構成で規定:
  - **第1号 = 募集適正基準**（寄附金の募集を適正に実施する地方団体であること）
  - **第2号 = 返礼割合3割基準**
  - **第3号 = 地場産品基準**
- これらの委任を受けて **平成31年総務省告示第179号** が実体基準を定める。地場産品基準の一部は地方税法施行規則にも規定。
- 告示第179号は **令和4年告示第203号・第323号、令和5年告示第244号、令和6年6月28日改正** 等で累次改正。

## 1. 募集適正基準（経費5割ルール）

**根拠**: 地方税法第37条の2第2項第1号・第314条の7第2項第1号 →（委任）告示第179号第2条第2号

**基準本体**: 指定対象期間において、第一号寄附金の**募集に要する費用の合計額**が、受領する寄附金の合計額の**百分の五十（50%）以下**であること。

### 「募集に要する費用」に含まれるもの

| 費用項目 | 補足 |
|---------|------|
| 返礼品等の調達費用 | ＝後述の3割基準の対象と同じもの |
| 返礼品等の送付（送料） | |
| 広告・宣伝費 | |
| ポータルサイト運営事業者へ支払う費用 | ポータル手数料・仲介手数料 |
| 決済費用 | クレジットカード決済手数料等 |
| 寄附金受領証（受領証明書）の発行事務費 | ← 2023年改正で算入明確化 |
| ワンストップ特例の申請書受付事務費 | ← 2023年改正で算入明確化 |
| 各種事務の委託費用 | |
| 職員の人件費 | ふるさと納税業務に係る分 |

→ これらの合計を寄附受領総額の50%以下に収める必要がある。
ポータル手数料（全国平均で寄附額の約11.5%）＋返礼品30%で、経費率はほぼ上限に張り付く構造。

### 2023年（令和5年）10月1日改正のポイント

- 改正前は受領証発行費用・ワンストップ特例事務費等の「付随費用」が5割計算の外という運用余地があった。
- 改正で**これら付随費用も含めて5割以下とすることを明確化・厳格化**。
- 結果、同じ寄附額でも返礼品等に充てられる原資が圧縮され、自治体は実質値上げで対応（件数微減・金額増の一因）。
- 同改正で地場産品基準も同時改正（熟成肉・精米の都道府県内産限定など）。

### 透明性確保の追加規定（令和7年6月24日改正）

- 1支払先あたり年100万円以上の募集費用について、支払先・支払額・支払目的の**公表**を求める規定を追加。

## 2. 返礼品3割ルール

**根拠**: 地方税法第37条の2第2項第2号・第314条の7第2項第2号 →（委任）告示第179号第3条

**基準本体**: **返礼品等の調達に要する費用**が、当該返礼品等の提供に係る寄附金の**百分の三十（30%）以下**であること。

### 「調達価格」の定義・運用

- 調達費用は、地方団体が**現に支出した額（実際の調達価格）**による。
- 返礼割合の判定は**個別の返礼品等ごと**に行い、**一時期であっても3割を超える返礼品等は不可**（瞬間的な超過も認められない）。
- 複数返礼品を一括調達する場合は、合計調達費用を数量・内容に応じて**按分**して各返礼品の調達価格を算定。

## 3. 地場産品基準

**根拠**: 地方税法第37条の2第2項第3号・第314条の7第2項第3号 →（委任）地方税法施行規則および告示第179号第5条

**基本理念**: 返礼品等が「当該地方団体の区域内において生産・製造・加工等されたもの」等であること。第5条が適合類型を号建てで列挙（号番号は累次改正で枝番増。以下は実務整理）。

| 類型 | 内容 |
|------|------|
| 第1号 | 区域内で**生産**されたもの |
| 第2号 | 区域内で**原材料の主要な部分が生産**されたもの |
| 第3号 | 区域内で**製造・加工等の主要な工程**を行い**相応の付加価値**が生じたもの（区域内加工の例外）|
| 第3号イ | 熟成肉・精米は**原材料が同一都道府県内産**のものに限る（令和5年10月改正で追加）|
| 第3号ロ | 区域内で企画立案等の主要工程が行われたもの |
| 第4号 | 流通構造上やむを得ず区域外品が混在する場合等の取扱い |
| 第5号 | PR目的で当該団体が製作したキャラクターグッズ・オリジナル品等で、団体独自と明白なもの |
| 第6号 | 地場産品と附帯品（区域外品）の組合せ。**地場産品の価値が全体の7割以上**の場合に限る（7割ルール）|
| 第7号（等）| 区域内で提供される役務（サービス）・宿泊等 |
| 第8号 | 区域内の地域資源で発電された電気 |
| 第9号 | 複数団体の**共通返礼品**（広域連携）、災害支援の代理寄附等の特例 |

### 例外規定の要点

- **区域内加工（第3号）**: 原材料が区域外でも、区域内の主要工程で相応の付加価値が生じれば可。ただし熟成肉・精米は同一都道府県内産原材料に限定。
- **共通返礼品（第9号）**: 単独では地場産品にならなくても、複数団体連携で各団体の地場産品を組み合わせる場合に許容。
- **附帯品との組合せ（第6号）**: 地場産品価値が7割以上であることが条件。

## 4. 富裕層所得制限（令和8年度税制改正 — 2028年度適用）

**根拠**: 令和8年度与党税制改正大綱（2025年12月19日決定）・2026年2月20日閣議決定

**概要**: 年収1億円を超える高額所得者に対し、住民税の特例控除額（ふるさと納税の控除）に**193万円の上限**を設ける。

| 項目 | 内容 |
|------|------|
| 対象 | 年収1億円超の高額所得者 |
| 上限額 | 特例控除額 193万円 |
| 適用開始 | 2028年度以後の住民税（2027年分の所得から） |
| 影響 | 寄附額の事実上の上限が引き下がる（現行は年収に応じて無制限に近い） |
