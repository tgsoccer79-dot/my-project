# 制度の概要・仕組み

> 最終更新: 2026-06-02
> 一次出典: 総務省「ふるさと納税ポータルサイト」・国税庁

---

## 1. ふるさと納税とは

「ふるさと納税」とは、好きな自治体に**寄附**をすることで、**寄附額から2,000円を差し引いた金額が、所得税・住民税から控除される制度**。名称は「納税」だが実態は**寄附**であり、使途や返礼品を選べる点が通常の税と異なる。

- 生まれた故郷や、応援したい自治体を自分で選べる
- 返礼品（特産品・宿泊券・体験等）を受け取れる
- 控除上限額の範囲内であれば、実質2,000円の負担で寄附できる

出典: 総務省 https://www.soumu.go.jp/main_sosiki/jichi_zeisei/czaisei/czaisei_seido/furusato/mechanism/deduction.html

## 2. 税控除の仕組み

控除は**所得税からの控除1本**と**住民税からの控除2本**の計3本立て。3本の合計が「寄附額 − 2,000円」になるよう設計されている。

### (1) 所得税からの控除（寄附金控除）

```
所得税控除額 = (寄附額 − 2,000円) × 所得税率
```

- 所得税率は課税所得に応じた超過累進税率（5〜45%）
- 対象上限: 総所得金額等の40%
- 翌年の確定申告で還付

### (2) 住民税からの控除：基本分

```
住民税控除額（基本分） = (寄附額 − 2,000円) × 10%
```

- 全国一律10%
- 対象上限: 総所得金額等の30%

### (3) 住民税からの控除：特例分

```
住民税控除額（特例分） = (寄附額 − 2,000円) × (100% − 10% − 所得税率)
```

ただし特例分が住民税所得割額の20%を超える場合は、**住民税所得割額 × 20%** が上限（←これが控除上限額の実質的な決まり方）。

### 「自己負担2,000円」の構造

3本の合計を整理すると：

```
所得税控除 + 住民税基本分 + 住民税特例分
= (寄附額−2,000) × 所得税率
+ (寄附額−2,000) × 10%
+ (寄附額−2,000) × (100% − 10% − 所得税率)
= 寄附額 − 2,000円
```

→ **控除上限額の範囲内であれば、手元から出ていくのは実質2,000円のみ**。

## 3. 控除上限額

### 上限の決まり方

「住民税特例分が住民税所得割額の20%以内」という条件から逆算：

```
寄附上限額 ≒ 住民税所得割額 × 20% ÷ (90% − 所得税率) + 2,000円
```

収入・家族構成で上限が変わる理由:
- 収入が高いほど住民税所得割額が大きく・所得税率も高いため上限が上がる
- 扶養控除・配偶者控除があると課税所得が減り上限が下がる

## 4. ワンストップ特例制度

確定申告をせずにふるさと納税の控除を受けられる簡便な手続き。

### 利用できる要件（3つすべて必要）

| 要件 | 内容 |
|------|------|
| ① 確定申告不要な人 | 給与所得者で、医療費控除・住宅ローン控除（初年度）等で確定申告を行わない方 |
| ② 寄附先が5自治体以内 | 同一年内の寄附先が5団体以下（同一自治体への複数回寄附は1団体と数える） |
| ③ 申請書の提出 | 各寄附先の自治体に「特例申請書」＋本人確認書類を郵送 |

### 申請期限

**寄附した翌年の1月10日（必着）**

- 期限を過ぎた・書類不備の場合は確定申告で対応（翌年2〜3月）
- 同年中に確定申告を行うとワンストップ申請は**すべて無効**になる

### ワンストップ特例 vs 確定申告

| 項目 | ワンストップ特例 | 確定申告 |
|------|----------------|----------|
| 対象者 | 給与所得者（確定申告不要者）のみ | すべての人 |
| 寄附先 | 5自治体以内 | 制限なし |
| 手続き | 申請書を各自治体へ郵送 | e-Tax or 税務署に提出 |
| 期限 | 翌年1月10日必着 | 翌年2月16日〜3月15日 |
| 控除方式 | 住民税からのみ全額控除（所得税還付なし） | 所得税（還付）＋住民税から控除 |
| 控除合計額 | 同額になるよう設計 | 同上 |

## 5. 寄附の流れ（手続き全体）

```
① ポータルサイト or 自治体HPで申込・寄附金を入金
        ↓
② 返礼品が到着（自治体・事業者から発送）
        ↓
③ 寄附金受領証明書が郵送で届く（繁忙期は数週間かかることあり）
        ↓
  【ワンストップ特例の場合】       【確定申告の場合】
  特例申請書＋本人確認書類を       受領証明書を添付して申告
  自治体へ郵送（1月10日必着）      （翌年2月16日〜3月15日）
        ↓                                  ↓
  翌年6月〜の住民税から控除        所得税：翌年4〜5月頃に還付
  （天引き減額）                   住民税：翌年6月〜から控除
```
