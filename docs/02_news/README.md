# ② ニュース・制度改定の追跡

最新の告示・報道を読み込み、解釈し、要約して周りに伝えられるようにする柱。

## このフォルダの構成

- [timeline.md](timeline.md) — 制度改定・主要ニュースの時系列ログ（一覧）
- [archive/](archive/) — 個別ニュース・告示の読み込みメモ（日付別ファイル）

## 運用ルール

1. 新しい告示・報道を見つけたら、`archive/YYYY-MM-DD_見出し.md` を作成。
2. 要点・背景・自分の解釈・影響範囲・出典URLを記録。
3. `timeline.md` に1行サマリ＋archiveへのリンクを追加。
4. 制度本体に関わる改定なら [01_seido/](../01_seido/) の該当ファイルにも反映。

## archiveメモのテンプレート

```markdown
# YYYY-MM-DD ◯◯（見出し）

- 出典: （媒体名・URL・取得日）
- 種別: 告示 / 報道 / 統計発表 / その他

## 要点（3行）
## 背景
## 解釈・影響（誰に・どう効くか）
## 関連リンク
```
