# 2026-06-24 令和7年総務省告示第220号 — 地場産品基準・募集費用透明化の改正

- 出典: 総務省報道資料（https://www.soumu.go.jp/menu_news/s-news/01zeimu04_02000144.html）・TKCエクスプレス（取得日 2026-06-08）
- 種別: 告示

## 要点（3行）

1. ご当地キャラクターグッズ等「広報目的返礼品」に直近1年の配布・販売実績要件と提供数量上限を明確化。
2. 年100万円以上の募集費用支払先（社名・金額・目的）を自治体が公表する義務を新設（2026年9月初回公表予定）。
3. いずれも**2026年10月開始の指定対象期間**から適用。告示番号：令和7年総務省告示第220号。

## 背景

- 「自治体ロゴを貼り付けただけ」の工業製品や、地域との関連が薄いご当地グッズが地場産品として認められてきた問題を是正。
- 大手ポータルへの手数料（年100万円以上）の不透明性が批判されており、透明化により市場の実態把握を促進する狙い。

## 改正の詳細

### 地場産品基準の明確化（第5号 PR目的グッズ）

| 要件 | 内容 |
|------|------|
| 実績要件 | 直近1年間（指定申請前1年）における区域内での**配布・販売実績**があること |
| 数量上限 | 指定対象期間における提供数量が、直近1年の実績数量の範囲内であること |
| 対象 | キャラクターグッズ、ロゴ入り商品等の広報目的品 |

### 募集費用の公表義務（透明性強化）

| 条件 | 内容 |
|------|------|
| 対象 | 1支払先あたり年間100万円以上の募集費用 |
| 公表事項 | 支払先の名称・支払額・支払目的 |
| 公表時期 | 2026年9月が初回公表予定 |

## 解釈・影響

- ご当地グッズの「在庫」や「過去実績のない新商品」を大量に返礼品として設定することが困難になる。
- 大手ポータルへの手数料が可視化されることで、手数料引き下げ交渉（2026年5月要請・8月回答期限）に加速の可能性。
- kokuji.md の透明性確保規定（令和7年6月24日改正）欄と対応。
