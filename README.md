# ふるさと納税 知識ベース（ふるさと納税博士プロジェクト）

ふるさと納税制度について「博士」レベルで体系的に理解・追跡・分析するための知識ベースです。
総務省HP・各ポータルサイト・ニュース・研究機関などの情報を集約し、Markdownで整理・更新していきます。

## このリポジトリの目的

1. **制度の体系知識** — 概要・歴史・法令・総務省告示を正確に押さえる
2. **ニュース／制度改定の追跡** — 最新の告示・報道を読み込み、解釈し、要約して伝えられるようにする
3. **市場・返礼品データ** — どのポータルでどの自治体のどの返礼品が売れているか、寄附額ランキングを整理する ⭐️ **最優先**
4. **自治体の経費・収支** — どの自治体がどんな経費でどれだけ消耗しているかを整理する

## 使い方・運用方針

- 各Markdownには **必ず出典（URL・取得日）** を明記する。
- 数値データは時点が命。`（YYYY年度／YYYY-MM-DD取得）` の形で時点を残す。
- 大きな更新ごとに意味のあるコミットメッセージで記録し、Gitで履歴を追えるようにする。
- まだ中身が空のセクションは「TODO」と明記し、埋めるべき項目をリスト化しておく。
